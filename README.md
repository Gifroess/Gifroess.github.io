# Currículo Online

Projeto desenvolvido durante a primeira semana da capacitação da área de Projetos Desktop da Asimov Jr.

O objetivo da atividade foi colocar em prática os conhecimentos adquiridos em um curso de Git e GitHub da Udemy, utilizando ferramentas de versionamento e hospedagem de código para personalizar e publicar um currículo online.

## Tecnologias utilizadas

- HTML
- CSS
- JavaScript
- Git
- GitHub
- GitHub Pages

## Aprendizados

Durante o desenvolvimento deste projeto, foram trabalhados conceitos como:

- Controle de versão com Git;
- Criação e gerenciamento de repositórios no GitHub;
- Commits, push e pull;
- Publicação de páginas web com GitHub Pages;
- Personalização de um template de portfólio online.
